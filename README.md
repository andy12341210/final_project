遊玩方式：

點擊主畫面可以輸入暱稱，採用之前災難公關的方式，不設置帳號，而是直接以暱稱加入。

點擊尋找房間

按選擇角色可以瀏覽角色並選擇

點擊準備，當房間滿人切都準備完成後開始遊戲

當輪到你的回合時，按左下角的骰子可以移動(右邊顯示輪到誰了)

遊玩15回合後遊戲結束

重新整理網頁能開啟新的一局(包含輸入暱稱)
